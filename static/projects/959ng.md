# The 959 Nematode Genomes Project (959NG)

**GoaT Search Term:** 959NG

# Scope: 

::grid[**959NG** is an open collaboration that exists to promote the sequencing of all nematode species. The number 959 is a placeholder for “a lot”. While most similar projects aim for a round “1,000”, we use an important nematological number for our headline aim: every adult female Caenorhabditis elegans nematode has exactly 959 somatic cells. For more information access the [959NG project page](959nematodes@sanger.ac.uk) or reach the contact info below.]{size=8}

**Bioproject ID** PRJEB81973

**Main Contact**: 959nematodes@sanger.ac.uk

:::grid{container direction="row" spacing="1" class="padded"}
:::

## 959NG Long List

959NG long_list correspond to the subset of species declared as targets by the project. To retrieve the list use the search terms:

- long_list=959NG
- tax_rank(species)
